# Tasks (đã sắp xếp theo flow tối ưu)

1. Làm đề:
   cho phép user chọn đề theo Level (1-3) và class classify (11-12-Ôn thi đại học).

2. nút learning -> dropdown:
   -> Roadmap Tutor: (1 trang giao tiếp với AI)
   lấy roadmap đã được phân/tạo của user để trao đổi với user.
   -> Học cùng AI:
   user vào trang -> hiển thị "Tính năng đang được update, các bạn vui lòng đón chờ nhé!".

3. Roadmap theo tiến độ:
   mỗi user roadmap có 5 module (AI tạo giáo trình/chuyên đề cho từng module, từ 1-5: cơ bản -> nâng cao, phù hợp level user);
   xong mỗi 2 module thì check tiến độ học tập.

4. Tiến độ user < tiến độ roadmap:
   hỏi user có theo kịp không;
   có -> học tiếp;
   không -> replanning với tiến độ chậm hơn.

5. Tiến độ user > tiến độ roadmap:
   hỏi user có muốn up-level không;
   có -> cho làm bài thi up-level;
   không -> học tiếp.
